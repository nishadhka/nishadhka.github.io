###with istsos dat formate - the istsos requieres date and time in this ( 2014-04-30T11:25:00.000000+0530) format, the data from dylos is cutshorten to reduce size. So to edit the column it has to converted into datframe in python. - Pandas data frame conversion based on this

col_names=["date","np05","np25"]
data = pa.read_csv("GVR.DAT", header=None,skiprows=1, parse_dates="date",names=col_names)

again reading the DAT file created in earlier step.

data["date"]=pa.to_datetime(data["date"])
data["date"] = data['date'].apply(lambda x: x.strftime("%Y-%m-%dT%H:%M:%S.000000+0530"))

calling specific date column, then converting into desired date formate.

###Time scaling

  • based on this and this
  • has to reindex the dataframe into datetimeindex as follows data2 = data.set_index(pa.DatetimeIndex(data['urn:ogc:def:parameter:x-istsos:1.0:time:iso8601']))
  • then using this index to resample dataframe for every fifteen minutes last data, default resample is ‘mean’ data3=data2.resample('15Min', how='last')
  • then set index to the date columns, other wise it write one extra column into dat file data4=data3.set_index('urn:ogc:def:parameter:x-istsos:1.0:time:iso8601')
  • then to make file out of this dataframe data4.to_csv('GVR1.DAT', sep=',', encoding='utf-8')
