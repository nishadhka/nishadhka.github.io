#to edit a table in postgresql

  • to list databases \list
  • to list tables in a database based on this SELECT table_schema,table_name FROM information_schema.tables ORDER BY table_schema,table_name;
  • to view details about a table \d table.name
  • to edit a column in a table ALTER TABLE cbed.measures ALTER COLUMN val_msr TYPE numeric(14,6);
