SCREEN usage

2014-01-16

SCREEN is a utility for multiple session in command line Linux. * To start a screen with the name screen -S “name without mark” * To view running screen screen -r * to get into a particular screen screen -xr pid(of the screen) * to get out of the screen CTRL+A+D
