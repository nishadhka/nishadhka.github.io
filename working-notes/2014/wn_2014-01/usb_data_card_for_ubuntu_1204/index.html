USB DATA card for ubuntu 1204

2014-01-16

  1. To detect the USB connected to system, Do a dmesg |grep tty
  2. If not showing anything, the issue was solved following final answer of this question http://ubuntuforums.org/archive/index.php/t-1853306.html

  3. dmesg is a “ring buffer, so if many messages are being logged, will lose the initial boot messages. Try this instead:”

    cd /var/log 
grep ttyUSB dmesg messages *log | more"

  4. now showing

    fellow@dhcppc3:~$ cd /var/log ; grep ttyUSB dmesg messages *log | more
