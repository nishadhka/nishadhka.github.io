Screen for remote head less sever

2013-11-05

Execute multiple programs simultaneously in the terminal, use the program screen. based on http://askubuntu.com/questions/163567/start-program-from-terminal and http://www.howtoforge.com/linux_screen

Basic running steps

  1. screen -r: to view any running screen
  2. If there is no running screen, by this command a new terminal will be open, and any commands can be run here, such as to get a file using sftp
  3. To get out of the screen type CTRL +A+D
  4. after this the cursor goes to the base terminal

based on http://askubuntu.com/questions/8653/how-to-keep-processes-running-after-ending-ssh-session http://askubuntu.com/questions/17211/how-to-send-sftp-file-download-to-background for working with SFTP http://www.usc.edu/its/sftp/cline.html
